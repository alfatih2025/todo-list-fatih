# 📝 Todo List Web Application

Aplikasi **Todo List** sederhana berbasis **HTML, CSS, dan JavaScript** yang memungkinkan pengguna mengelola daftar tugas secara mudah, cepat, dan interaktif. Project ini cocok sebagai latihan dasar pengembangan web frontend maupun sebagai pondasi untuk aplikasi manajemen tugas yang lebih kompleks.

---

# 📖 Deskripsi

Todo List merupakan aplikasi web sederhana yang dirancang untuk membantu pengguna mencatat, mengelola, dan memantau tugas harian.

Project ini dibangun menggunakan teknologi frontend murni tanpa framework sehingga mudah dipahami oleh pemula yang sedang belajar pengembangan web.

---

# ✨ Fitur

- ➕ Menambahkan tugas baru
- 📋 Menampilkan daftar tugas
- ✅ Menandai tugas yang telah selesai
- ❌ Menghapus tugas
- ⚡ Tampilan ringan dan responsif
- 🌐 Berjalan langsung di browser tanpa instalasi

---

# 🛠 Teknologi

- HTML5
- CSS3
- JavaScript (Vanilla JS)

# 🚀 Cara Menjalankan

## 1. Clone Repository

```bash
git clone https://github.com/username/todo-list-fatih.git
```

atau download dalam bentuk ZIP.

---

## 2. Buka Project

Masuk ke folder project kemudian buka file

```
index.html
```

menggunakan browser seperti:

- Google Chrome
- Microsoft Edge
- Mozilla Firefox
- Brave

Tidak memerlukan server maupun database.

---

# ⚙ Cara Penggunaan

1. Ketik tugas pada kolom input.
2. Klik tombol **Tambah**.
3. Tugas akan muncul pada daftar.
4. Klik tugas untuk menandainya sebagai selesai (jika tersedia).
5. Klik tombol hapus untuk menghapus tugas dari daftar.
